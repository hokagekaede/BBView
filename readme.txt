■不具合
・BB.NETデータ取得時、IC名が同名の場合にデータを読み込めない
・BB.NETのIC名にHTMLタグで使用する文字が含まれていた場合、想定外の表示になる。
・アセン共有が公式Twitterアプリで読み込めない。

■改善案
[パーツ選択画面]
・軽量型/標準型/重量型でのフィルタ機能
・二脚/ホバーでのフィルタ機能
・任意のシリーズの位置にジャンプする機能

[チップ選択画面]
・チップでの上昇値の表示

・ダッシュ時/加速時/AC使用時の総移動距離の表示

・上限超過したスペックの警告表示
・状態設定に72AN手榴弾被弾時を追加(装甲低下)

[耐性画面]
・シミュレーション機能化

[購入プレビュー画面]
・購入可能なアイテムに色を付ける。

[店舗検索画面]
・トップページに店舗ショートカットを付ける。
・トップページに地図を表示して、都道府県(または地方)を選択しやすくする。
